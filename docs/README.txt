======================================================================
Super Copa Perú Evolution
======================================================================

INSTALLATION
----------------------------------------------------------------------
1. Install prerequisites:
   - CMake 3.15+
   - C++17 compiler (GCC, Clang, MSVC)
   - Git
   - SFML 3.1 (fetched automatically)

2. Clone and build:
   git clone https://github.com/blayne-04/SPCE.git
   cd SPCE
   cmake -S . -B build
   cmake --build build

3. The executable will be in: build/bin/

4. Run the game from the project root folder:
   ./build/bin/SPCE

HOW TO PLAY
----------------------------------------------------------------------
Goal: Score more goals than the opponent in 3 minutes.

Controls:
  Move player ........ WASD
  Pass ............... J
  Shoot .............. K
  Tackle / Steal ..... L
  Switch player ...... I

- Possession changes automatically after tackles or goals.
- Cows may randomly invade the pitch – they block shots and disrupt passes.
