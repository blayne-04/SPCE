# 🚀 Pro-level suggestion (worth it)

Instead of hardcoding:

### Build a helper:

```cpp
sf::IntRect tileRect(int tx, int ty, int w, int h) {
    return sf::IntRect(tx * 16, ty * 16, w * 16, h * 16);
}
```

Now everything becomes clean:

```cpp
auto centerCircle = tileRect(10, 0, 6, 6);
auto leftPenalty  = tileRect(10, 6, 6, 8);
```
